#### **2.1 解析器接口（parsers/base_parser.py）**
```python
from abc import ABC, abstractmethod
from typing import Dict, List, Optional, Any
from dataclasses import dataclass

@dataclass
class ParserResult:
    """解析结果标准化数据类"""
    exam_title: str                    # 试卷标题
    subject: Optional[str] = None      # 科目
    level: Optional[str] = None        # 等级
    questions: List[Dict] = None       # 题目列表
    total_points: int = 100            # 总分
    has_images: bool = False           # 是否包含图片
    images_data: List[Dict] = None     # 图片数据（base64、位置等）
    metadata: Dict[str, Any] = None    # 额外元数据

class BaseParser(ABC):
    """解析器抽象基类 - 所有解析器必须实现的接口"""
    
    @abstractmethod
    def parse(self, file_path: str) -> ParserResult:
        """
        解析文档，返回标准化的解析结果
        
        Args:
            file_path: 文档文件路径
            
        Returns:
            ParserResult: 包含试卷标题、题目列表等信息的标准化结果
            
        Raises:
            FileNotFoundError: 文件不存在
            ParserError: 解析失败（自定义异常）
            ValueError: 文件格式不支持
        """
        pass
    
    @abstractmethod
    def supported_extensions(self) -> List[str]:
        """返回支持的文档扩展名列表，如 ['.docx', '.pdf']"""
        pass
    
    @abstractmethod
    def validate_file(self, file_path: str) -> bool:
        """验证文件是否可被解析，返回布尔值"""
        pass
```

```python
from abc import ABC, abstractmethod
from typing import Tuple, Dict, Any, Optional
from enum import Enum

class QuestionType(Enum):
    """题目类型枚举"""
    SINGLE_CHOICE = "single_choice"    # 单选题
    MULTIPLE_CHOICE = "multiple_choice" # 多选题
    JUDGMENT = "judgment"              # 判断题
    FILL_BLANK = "fill_blank"          # 填空题
    SUBJECTIVE = "subjective"          # 主观题
    PROGRAMMING = "programming"        # 编程题

class BaseGrader(ABC):
    """判卷器抽象基类 - 支持多题型判卷"""
    
    @abstractmethod
    def grade(
        self,
        student_answer: str,
        correct_answer: str,
        question_type: QuestionType,
        question_data: Optional[Dict] = None
    ) -> Tuple[float, bool, Dict[str, Any]]:
        """
        判卷核心方法
        
        Args:
            student_answer: 学生答案
            correct_answer: 正确答案
            question_type: 题目类型
            question_data: 额外题目数据（选项、分值等）
            
        Returns:
            Tuple[得分(0-100), 是否正确, 判卷详情]
        """
        pass
    
    @abstractmethod
    def normalize_answer(
        self,
        answer: str,
        question_type: QuestionType
    ) -> str:
        """标准化答案（去除空格、统一大小写等）"""
        pass
    
    @abstractmethod
    def get_supported_types(self) -> List[QuestionType]:
        """返回支持的题目类型列表"""
        pass
```

#### **2.3 服务层基类（services/base_service.py）**
```python
from abc import ABC
from typing import Optional, TypeVar, Generic
from flask_sqlalchemy import SQLAlchemy

T = TypeVar('T')

class BaseService(ABC, Generic[T]):
    """服务层基类 - 提供通用的CRUD和业务逻辑封装"""
    
    def __init__(self, db: SQLAlchemy, model_class: T):
        self.db = db
        self.model_class = model_class
    
    def create(self, **kwargs) -> T:
        """创建记录"""
        instance = self.model_class(**kwargs)
        self.db.session.add(instance)
        self.db.session.commit()
        return instance
    
    def get_by_id(self, id: int) -> Optional[T]:
        """根据ID获取记录"""
        return self.model_class.query.get(id)
    
    def update(self, id: int, **kwargs) -> Optional[T]:
        """更新记录"""
        instance = self.get_by_id(id)
        if instance:
            for key, value in kwargs.items():
                setattr(instance, key, value)
            self.db.session.commit()
        return instance
    
    def delete(self, id: int) -> bool:
        """删除记录"""
        instance = self.get_by_id(id)
        if instance:
            self.db.session.delete(instance)
            self.db.session.commit()
            return True
        return False
    
    def list_all(self, **filters) -> List[T]:
        """获取所有记录，支持筛选"""
        query = self.model_class.query
        for key, value in filters.items():
            if hasattr(self.model_class, key):
                query = query.filter(getattr(self.model_class, key) == value)
        return query.all()
```


#### **关键依赖关系详情**

1. **routes → services**：视图层调用服务层处理业务逻辑
2. **services → models**：服务层操作数据库模型进行数据持久化
3. **services → parsers**：试卷服务调用解析器处理文档
4. **services → utils**：服务层使用工具函数（验证、格式化等）
5. **parsers → utils**：解析器使用图片处理、文件验证工具
6. **所有层 → utils/error_handlers**：统一错误处理机制

#### **数据流示例：试卷上传流程**
```
1. routes/exam.py (视图层) 接收文件上传请求
   ↓
2. 调用 services/exam_service.upload_exam()
   ↓
3. exam_service 根据策略选择 parsers/word_parser 或 parsers/ai_parser
   ↓
4. 解析器调用 utils/image_utils 提取图片
   ↓
5. exam_service 调用 models/ 创建 Exam 和 Question 记录
   ↓
6. 返回标准化结果给视图层
   ↓
7. 视图层使用 utils/response_utils 包装响应
```

#### **详细表结构**

**1. Subject（科目表）**
```
subject_id: INTEGER (PK, AUTOINCREMENT)
name: VARCHAR(50) NOT NULL UNIQUE  # 科目名称（Python/C++/Java）
description: TEXT                  # 科目描述
created_at: DATETIME DEFAULT CURRENT_TIMESTAMP
updated_at: DATETIME DEFAULT CURRENT_TIMESTAMP
```

**2. Level（等级表）**
```
level_id: INTEGER (PK, AUTOINCREMENT)
name: VARCHAR(50) NOT NULL UNIQUE  # 等级名称（一级/二级/三级）
description: TEXT                  # 等级描述
created_at: DATETIME DEFAULT CURRENT_TIMESTAMP
updated_at: DATETIME DEFAULT CURRENT_TIMESTAMP
```

**3. Exam（试卷表）**
```
exam_id: INTEGER (PK, AUTOINCREMENT)
title: VARCHAR(200) NOT NULL        # 试卷标题
subject_id: INTEGER (FK → Subject.subject_id)
level_id: INTEGER (FK → Level.level_id)
total_points: INTEGER DEFAULT 100   # 总分
question_count: INTEGER DEFAULT 0   # 题目数量
has_images: BOOLEAN DEFAULT FALSE   # 是否包含图片
file_path: VARCHAR(500)             # 原始文件路径
created_at: DATETIME DEFAULT CURRENT_TIMESTAMP
updated_at: DATETIME DEFAULT CURRENT_TIMESTAMP

索引：
- idx_exam_subject_level (subject_id, level_id)
- idx_exam_created_at (created_at DESC)
```

**4. Question（题目表）**
```
question_id: INTEGER (PK, AUTOINCREMENT)
exam_id: INTEGER (FK → Exam.exam_id, ON DELETE CASCADE)
type: VARCHAR(20) NOT NULL          # 题型（single_choice/multiple_choice等）
text: TEXT NOT NULL                 # 题目文本
options: JSON                       # 选项列表 [{id: 'A', text: '...'}, ...]
correct_answer: TEXT NOT NULL       # 正确答案
points: INTEGER DEFAULT 10          # 分值
order_num: INTEGER                  # 题目顺序
has_image: BOOLEAN DEFAULT FALSE    # 是否包含图片
image_data: TEXT                    # 图片base64或路径
metadata: JSON                      # 额外元数据
created_at: DATETIME DEFAULT CURRENT_TIMESTAMP

索引：
- idx_question_exam (exam_id)
- idx_question_type (type)
- idx_question_order (exam_id, order_num)
```

**5. Submission（答题提交表）**
```
submission_id: INTEGER (PK, AUTOINCREMENT)
exam_id: INTEGER (FK → Exam.exam_id)
student_id: VARCHAR(50)             # 学生标识（扩展后可关联用户表）
student_name: VARCHAR(100)          # 学生姓名
total_score: DECIMAL(5,2)           # 总分
submit_time: DATETIME DEFAULT CURRENT_TIMESTAMP
duration_seconds: INTEGER           # 答题时长（秒）
is_completed: BOOLEAN DEFAULT TRUE  # 是否完成
metadata: JSON                      # 提交元数据

索引：
- idx_submission_exam (exam_id)
- idx_submission_student (student_id)
- idx_submission_time (submit_time DESC)
```

**6. Answer（答题记录表）**
```
answer_id: INTEGER (PK, AUTOINCREMENT)
submission_id: INTEGER (FK → Submission.submission_id, ON DELETE CASCADE)
question_id: INTEGER (FK → Question.question_id)
student_answer: TEXT                # 学生答案
score: DECIMAL(5,2)                 # 本题得分
is_correct: BOOLEAN                 # 是否正确
grading_details: JSON               # 判卷详情
answered_at: DATETIME DEFAULT CURRENT_TIMESTAMP

索引：
- idx_answer_submission (submission_id)
- idx_answer_question (question_id)
- idx_answer_correct (is_correct)
```

#### **外键约束**
- `Exam.subject_id` → `Subject.subject_id`
- `Exam.level_id` → `Level.level_id`
- `Question.exam_id` → `Exam.exam_id` (ON DELETE CASCADE)
- `Submission.exam_id` → `Exam.exam_id`
- `Answer.submission_id` → `Submission.submission_id` (ON DELETE CASCADE)
- `Answer.question_id` → `Question.question_id`
