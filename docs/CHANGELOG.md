## 版本说明

### 语义化版本规则
- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能新增
- **修订号**：向下兼容的问题修正

### 发布周期
- **主版本**：重大架构变更或功能重构
- **次版本**：新功能发布
- **修订版**：Bug 修复和小改进
