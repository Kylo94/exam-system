## 文档使用建议

### 首次部署
1. 阅读 `部署指南.md`
2. 按照 `Docker快速部署指南.md` 进行部署
3. 使用 `部署检查清单.md` 验证部署

### 生产环境
1. 阅读 `生产环境部署指南.md`
2. 参考 `配置检查报告.md` 检查配置
3. 使用 `生产环境测试说明.md` 进行验证

### 问题排查
1. 查看各文档中的故障排查章节
2. 参考 `Docker部署问题修复总结.md`
3. 查看日志文件

### 数据库管理
1. 参考 `数据库迁移指南.md`
2. 使用备份脚本定期备份
