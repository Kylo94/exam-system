## 🎯 使用指南

### 首次部署

如果你是第一次部署系统，请按照以下顺序阅读文档：

1. **开始**：阅读 [部署指南](部署指南.md) 的"快速开始"部分
2. **配置**：按照部署指南配置环境变量
3. **部署**：执行部署命令
4. **验证**：使用 [部署检查清单](部署检查清单.md) 验证部署

### 本地开发

如果你是开发人员，建议阅读：

1. [部署指南](部署指南.md) - 本地开发部署章节
2. [数据库迁移指南](数据库迁移指南.md) - 数据库管理
3. [日志系统说明](日志系统说明.md) - 日志配置

### 生产部署

如果你要部署到生产环境，建议阅读：

1. [部署指南](部署指南.md) - 生产环境部署章节
2. [部署检查清单](部署检查清单.md) - 完整检查清单
3. [生产环境部署指南](生产环境部署指南.md) - 详细配置
4. [生产环境测试说明](生产环境测试说明.md) - 测试验证

### 功能学习

如果你想了解系统功能，建议阅读：

1. [教师学生角色系统说明](教师学生角色系统说明.md) - 角色权限
2. [AI功能使用说明](AI功能使用说明.md) - AI 功能
